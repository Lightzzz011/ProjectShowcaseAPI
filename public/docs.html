ProjectShowcase API — Docs

Quick examples and usage. Base URL: https://projectshowcaseapi.onrender.com

GET /api/v1/projects

List projects. Query params: q, tag, page, perPage, sort.
curl "https://projectshowcaseapi.onrender.com/api/v1/projects?q=queue&tag=Python&page=1&perPage=5"
JS (fetch)
fetch("/api/v1/projects?q=queue&tag=Python")
  .then(r => r.json())
  .then(d => console.log(d));

GET /api/v1/projects/{id}

Get full project details by id (e.g., proj-1).
curl "https://projectshowcaseapi.onrender.com/api/v1/projects/proj-1"
JS (fetch)
fetch("/api/v1/projects/proj-1").then(r=>r.json()).then(d=>console.log(d))

POST /api/v1/contact

curl -X POST "https://projectshowcaseapi.onrender.com/api/v1/contact" -H "Content-Type: application/json" -d '{"name":"Sai","email":"sai@example.com","message":"Hello"}'
fetch("/api/v1/contact", {method:"POST",headers:{"Content-Type":"application/json"},body:JSON.stringify({name:"Sai",email:"sai@example.com",message:"Hello"})}).then(r=>r.json()).then(d=>console.log(d))
